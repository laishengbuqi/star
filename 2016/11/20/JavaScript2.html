JavaScript 变量

命名

  1. 方法
    • 匈牙利命名法 变量名 = 类型 + 描述
      • int 类型 i
      • float 类型 fl
      • boolean 类型 b
      • string 类型 s
      • array 类型 a
      • object 类型 o
      • function 类型 fn
      • RegExp 类型 re
    • 驼峰命名法
      • 全部大写 (单词与单词之间用下划线分割)
      • 大小写混合 (第一个单词首字母小写,其他首字母大写)
  2. 规则
    • 首字符(英文字母或下划线)
    • 组成(英文字母、数字、下划线)
    • 禁忌( JavaScript 关键字、保留字)

声明

  1. 显示声明
    • var 变量名
  2. 陋习
    • 没有类型
    • 重复声明
    • 隐式声明
    • 不声明直接复制
  3. 正解
    • 先声明,后续写
    • 先赋值,后运算

变量类型

  1. 值类型
    • 占用空间固定,保存在栈中
    • 保存与复制的是值本身
    • 可使用typeof获取数据的类型
    • 基本数据类型都是值类型
  2. 引用类型
    • 占用空间不固定,保存在堆中
    • 保存与复制的是指向对象的指针
    • 使用instanceof检测数据的类型
    • 使用new操作实例化的对象,内置对象以及对象字面量方法创建的对象都是引用类型

作用域

  1. 全局变量
    • 包含
      • 在函数外部定义的变量
      • 在函数内部定义的无var声明的变量
    • 调用
      • 任何位置
  2. 局部变量
    • 包含
      • 在函数体内部使用var声明的变量
      • 函数的参数变量
    • 调用
      • 当前函数体内部
  3. 优先级
    • 局部变量高于同名全局变量
    • 参数变量高于同名全局变量
    • 局部变量高于同名参数变量
  4. 特性
    • 无块级作用域(ES6中let、const)
    • 全局变量是全局对象的属性
    • 局部变量是调用对象的属性
    • 作用域链
      • 内层函数可访问外层函数的局部变量
      • 外层函数无法访问内层函数的局部变量
    • 声明周期
      • 全局变量 (除非被显式删除,否则一直存在)
      • 局部变量 (紫声明起至函数执行完毕或被显式删除)
      • 回收机制
        1. 标记计数法
        2. 引用计数法
