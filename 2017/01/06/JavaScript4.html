JavaScript 数组

创建方法

  1. 空数组 var arr = new Array();
  2. 指定长度 var arr = new Array(size);
  3. 指定元素 var arr = new Array(item1,item2,item3,...itemN);
  4. 单维数组 var arr = [item1,item2,item3,...itemN];
  5. 多维数组 var arr = new Array ([数组序列1],[数组序列2],[数组序列N]);

基本操作

  1. 存储数组元素

    • 单维数组 数组名[下标索引]
    • 多维数组 数组名[外层数组下标][内层数组下标]
    • 特性
      • 数组长度是弹性的,可自由伸缩
      • 数组下标从0开始
      • 下标类型
        • 数值
        • 非数值
          • 转为字符串
          • 生成关联数组
          • 下标作为对象属性的名字
      • 数组元素可添加到对象中

  2. 添加数组元素

    • 使用 “[]” 运算符指定一个新下标
  3. 删除数组元素
    • delete 数组名[下标] 数组长度不变
  4. 遍历数组
    • for(var 数组元素下标 in 数组)
    • for(var i=0,l=数组.length;i<l;i++)

数组属性

  1. constructor 引用数组对象的构造函数 , 数组元素可添加到对象中
  2. length 返回数组的长度
  3. prototype 数组对象的原型 , 通过添加属性和方法来扩展数组的定义

ECMAScript3 方法

  1. 添加
    • push() 在数组末尾添加元素
    • unshift() 在数组头部添加元素
    • concat() 合并数组 返回一个新数组
  2. 删除
    • pop() 删除并返回数组的最后一个元素
    • shift() 删除并返回数组的第一个元素

  3. 子数组

    • splice()
      • 功能
        • 删除任意数量的项
          1. 要删除的起始下标
          2. 要删除的项数
        • 在指定位置插入指定项
          1. 起始下标
          2. 0 (不删除任何项)
        • 替换任意数量的项
          1. 起始下标
          2. 要删除的项数
          3. 要插入的项
          4. 插入项数和删除项数一样即可实现数组元素替换
      • 返回值 数组(包含被删除的元素)
    • slice()
      • 功能
        • 从已有数组中选取部分元素构成新数组
      • 参数
        • 返回项的起始位置
        • 返回项的结束位置
      • 特性
        • 如果是负数,则用数组长度加上该值确定位置
        • 起始位置实为数组的实际下标
        • 结束位置的实际下标为结束数值减1

  4. 数组排序

    • reverse() 颠倒数值中元素的顺序

    • sort()

      • 功能 : 对字符数组或数值数组进行排序

      • 特性:

        1. 默认为按照字符串比较

        2. 按数值大小比较需函数支持

          • 升序

          • 降序

  5. 数组转换

    • toString() 转换为字符串并返回
    • toLocaleString() 转换为本地格式字符串并返回
    • join() 用指定分割符合并数组每个元素并转换为字符串返回

ECMAScript5 方法

  1. 位置方法
    • indexOf 从数组的起始位置开始查找指定元素,若找到,返回指定元素在数组中的下标,否则返回-1
    • lastIndexOf() 从数组的结束位置开始查找指定元素,若找到,返回指定元素在数组中的下标,否则返回-1
    • 参数 1. 要查找项 2. 表示查找起点的位置索引
  2. 迭代方法
    • every() 如果该函数对每一项都返回true,则返回true
    • filter() 返回值为true的所有数组成员
    • forEach() 无返回值
    • map() 返回每次函数调用的结果数组
    • some() 有任意一项返回true,则返回true
    • 概要
      • 接收参数
        1. 要在每一项上运行的函数
        2. 运行该函数的作用域对象
      • 传入参数
        1. 数组项的值item
        2. 该项在数组中的位置索引index
        3. 数组对象本身array
  3. 缩小方法
    • reduce() 从数组起始位置开始遍历
    • reduceRight() 从数组末尾开始遍历
    • 参数
      • 接收参数
        1. 每一项上调用的函数
        2. 作为缩小基础的初始值
      • 传入参数
        1. 前一个值prev
        2. 当前值 cur
        3. 项的索引 index
        4. 数组对象 Array
